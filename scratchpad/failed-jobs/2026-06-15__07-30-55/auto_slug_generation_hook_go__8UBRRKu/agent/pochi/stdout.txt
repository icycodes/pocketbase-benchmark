## What was implemented

**`/home/user/myproject/main.go`** — a complete PocketBase embedding with a `posts` create hook:

```go
app.OnRecordCreateRequest("posts").BindFunc(func(e *core.RecordRequestEvent) error {
    title := e.Record.GetString("title")
    if title == "" {
        return apis.NewApiError(http.StatusBadRequest, "title is required", nil)
    }
    slug := slugify(title)
    e.Record.Set("slug", slug)
    return e.Next()
})
```

### Key decisions

| Concern | Detail |
|---|---|
| **Hook** | `OnRecordCreateRequest("posts")` — fires before the record is persisted, scoped to the `posts` collection |

| **Chain propagation** | `return e.Next()` at the end ensures the record creation continues normally |
