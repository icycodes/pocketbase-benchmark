### `src/hooks.server.js` — Server hook (middleware)
```
PocketBase instance  →  loadFromCookie()  →  authRefresh()  →  resolve()  →  exportToCookie()
```
- Creates a **fresh `PocketBase` instance per request** pointing to `http://127.0.0.1:8090`.
- Calls `pb.authStore.loadFromCookie()` with the incoming `Cookie` header.
- If the store is valid, calls `authRefresh()` to revalidate the token; clears the store on failure.
- Attaches the instance to `event.locals.pb` so all routes share it.
- After resolving, appends the serialized auth state via `set-cookie` using `exportToCookie()`.
