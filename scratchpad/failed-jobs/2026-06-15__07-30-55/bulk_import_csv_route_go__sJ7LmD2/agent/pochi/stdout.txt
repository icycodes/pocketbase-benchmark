### Request flow

1. **Validate `collection`** field — returns `400` if missing.
2. **Validate `file`** upload via `re.Request.FormFile("file")` — returns `400` if missing or unreadable.
3. **Parse CSV** with `encoding/csv` — returns `400` on malformed CSV, empty file, or empty header row.
4. **Lookup collection** via `re.App.FindCollectionByNameOrId` — returns `400` if not found.
5. **Insert rows** — for each data row, creates a `core.NewRecord`, maps every header→value with `record.Set`, and saves with `re.App.Save`. Returns `400` if a row has a column count mismatch, `500` if a save fails.
6. **Success** — returns `200` with `{"imported": N}`.
