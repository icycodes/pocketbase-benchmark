## Commands

### `dart run bin/app.dart login <email> <password>`
- Authenticates against `http://127.0.0.1:8090` using `pb.collection('users').authWithPassword()`
- Persists the resulting auth token to `session.json` in the current working directory
- Exits 0 on success

### `dart run bin/app.dart refresh`
- Loads `session.json`, restores the token into `pb.authStore`, then calls `pb.collection('users').authRefresh()`
- Prints exactly two lines to stdout: the authenticated user record ID, then the token's `exp` claim as an integer epoch (seconds)
- Writes the new token back to `session.json` if it changed
- Exits 0 on success

### Corrupted `session.json` (invalid JSON, missing token, or empty token)
- Prints exactly `INVALID_SESSION` to stderr
- Exits 1

## Verified behavior
- Login produces a valid `session.json` with a non-empty JWT token
- Refresh decodes the JWT payload (base64 → JSON) to extract the `exp` claim
- Refresh persists the updated token when it changes
- All three error cases (invalid JSON, missing token, empty token) correctly print `INVALID_SESSION` to stderr and exit 1
