I have created the PocketBase JavaScript event hook to sanitize the `email` field for the `users` collection. 

The script is saved as `/home/user/pocketbase/pb_hooks/sanitize_email.pb.js` and intercepts both `onRecordCreateRequest` and `onRecordUpdateRequest` events. 

It retrieves the `email` field from the record, trims any leading or trailing whitespace, and converts all characters to lowercase before saving the changes to the database and continuing the request chain.

Here is the implementation of the hook:
```javascript
onRecordCreateRequest((e) => {
    let email = e.record.get("email");
    if (email) {
        e.record.set("email", email.trim().toLowerCase());
    }
    e.next();
}, "users");

onRecordUpdateRequest((e) => {
    let email = e.record.get("email");
    if (email) {
        e.record.set("email", email.trim().toLowerCase());
    }
    e.next();
}, "users");
```

I have also verified that the hook works correctly via the API for both POST (create) and PATCH (update) requests.
