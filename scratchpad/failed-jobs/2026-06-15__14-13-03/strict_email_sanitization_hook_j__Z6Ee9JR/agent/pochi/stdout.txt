Created `/home/user/myproject/pb_hooks/sanitize_email.pb.js` with the following hook:

```js
onRecordCreateRequest((e) => {
    const email = e.record.get("email");

    if (typeof email === "string") {
        e.record.set("email", email.trim().toLowerCase());
    }

    e.next();
}, "users");
```

**How it works:**

- `onRecordCreateRequest` — intercepts every `POST /api/collections/users/records` request before the record is persisted.
- The second argument `"users"` scopes the hook exclusively to the `users` collection.
- `e.record.get("email")` reads the raw email value submitted by the client.
- `.trim().toLowerCase()` removes surrounding whitespace and normalises case.
- `e.record.set("email", ...)` writes the sanitized value back onto the record object.
- `e.next()` propagates execution through the remaining middleware chain so the record creation proceeds normally.

PocketBase auto-loads all `*.pb.js` files from `pb_hooks/` on startup, so no additional registration is required. Start the server with:

```
./pocketbase serve --http="0.0.0.0:8090"
```
